# Power BI Sales Dashboard 📊

## 📌 Project Overview

This project contains an interactive **Power BI dashboard** designed to analyze and visualize sales data. The dashboard helps users understand business performance, identify trends, and make data-driven decisions.

The report provides insights into sales metrics using clear and interactive visualizations.

## 🎯 Objectives

* Analyze overall sales performance
* Identify trends and patterns in sales data
* Provide clear visual insights for decision making
* Build an interactive business intelligence dashboard

## 📂 Project File

* `parth_powerbi.pbix` – Power BI dashboard file

## 📊 Dashboard Features

* Total Sales overview
* Key performance indicators (KPIs)
* Interactive data filters
* Sales trend analysis
* Visual reports using charts and cards
* Dynamic Power BI visuals

## 🛠️ Tools & Technologies

* **Power BI**
* **Data Visualization**
* **Business Intelligence**
* **Data Analysis**

## 🚀 How to Use

1. Download the `.pbix` file from this repository.
2. Open the file in **Microsoft Power BI Desktop**.
3. Explore the interactive dashboard and visual reports.
